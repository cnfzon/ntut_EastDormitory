# ntut_EastDormitory
北科東校區宿舍網站
## 專案簡介

本專案為一個宿舍委員會專用的資訊網站，提供以下功能：
- 宿舍環境介紹（A 棟 / B 棟）
- 宿委會幹部介紹
- 問題回報與意見收集（整合 Google 表單）
- 最新公告區塊（連接 MongoDB 資料庫）
- 導航清晰、行動裝置響應式設計
